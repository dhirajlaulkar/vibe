# Vibe

An AI-powered app builder similar to [Lovable](https://lovable.dev/). Vibe empowers you to create full-stack web applications simply by describing them, leveraging advanced AI to generate code, preview implementations, and iterate on your ideas instantly.

## Features

- **AI-Driven Development**: Describe your app idea in natural language and let the AI build it.
- **Instant Preview**: See your changes live as the code is generated.
- **Modern Stack**: Built with Next.js for high performance and scalability.

## Getting Started

First, run the development server:

```bash
npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
